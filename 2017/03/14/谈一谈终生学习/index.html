谈一谈终生学习

发表于 2017-03-14 | 分类于 感悟 |
“得到”APP改版和“罗辑思维”第五季改版带来的思考

步入而立之后，我的求知欲是“罗胖”点燃的。

2014年5月，我开始戒烟；同年9月，开始慢跑。

于我而言，2014年是人生的一个转折点，因为我开始真正能理解和掌控自己的欲望了。也正是这个时期，不经意之间邂逅了当时还是轻量级产品的“罗辑思维”。当时罗胖搬运的各种新鲜观点震得我一愣一愣的，于是我几乎每期不拉地追完了他的节目，每本不拉地买了他推荐的书，还买会员啊，入社群啊，搞分享啊，整得不亦乐乎，以为这就是世界的全部意义了，直到我开始了解到有一个节目叫“晓松奇谈”……

追了几期“晓松奇谈”之后，我慢慢发现，虽然世界不是教科书说的那样，可也不仅仅是罗胖描述的那样，还有晓松老师介绍的那样。于是我就纳闷了，为啥同一个世界，描述起来却是那么不一样呢？

于是我开始不满足于教科书、罗胖和晓松老师给我看的世界，我开始自己看书，胡乱地看。

看完了易中天先生的《中华史》，突然就对《世界史》感兴趣，胡乱看了一些内容之后，就自己画了好几张历史坐标图，还写了一篇6000字的文章……看过了K·K著名的《失控》之后，就顺便看完了《科技想要什么》、《技术元素》和《必然》，于是很好奇K·K为什么有这种想法，就按图索骥，根据他某本书上的介绍，认真学习了理查德·道金斯的《自私的基因》，有感于生命的神奇，于是又对人类整个历史特别感兴趣，又看起了尤瓦尔·赫拉利的《人类简史》，顺便也就看了《未来简史》……
在这个过程中有时候觉得这些讲道理的书好无聊啊，于是又陆陆续续看了很多侦探小说，还硬着头皮啃完了《百年孤独》，可是直到把《霍乱时期的爱情》看到一半的时候，我才稍微有点感触到文学的魅力，以及加西亚·马尔克斯为什么能拿诺贝尔奖……
兴趣的真理线程突然启动，于是买了很多物理的科普书，可一翻发现连最基础的都看不懂，所以重新买了高中物理教科书，打算从头看起……

这就是这两年多来本人混乱的学习过程。

没有主线，全凭兴趣的学习，虽然偶尔也能拾人牙慧随口掰扯几个观点糊弄糊弄那些吃瓜群众们，但毕竟虚啊！知乎上的大V们说，学习的目标是 构建自己的知识体系，原本我满以为书看多了自然也就会了，现在真的是看了一点点书了，发现真不是这样。

不花点水磨工夫一点点地 把社会科学、自然科学和人文艺术的基础框架打好，那些看过的书就跟空中楼阁一般无处安放，最后只能散落一地混成一堆，只能指着那些似是而非的理论哑口无言。回想起来，这些道理并不是第一天知道的，其实一上学老师就说了的，只不过年轻时荷尔蒙旺盛，光顾着吸引女孩子注意了，谁tm记得老师说什么了呢。

好在现在有点买书的钱、有闲（不去应酬谁都有闲），思想觉悟也到了，好好补上这一课的机会也就来了。接下来的日子里，从物理开始，然后把化学、生物这些学生阶段避之唯恐不及的科目逐一学回来，先用一两年的时间把知识体系搭建起来，而后才谈得上 学习 这两个字。

这就是我对终生学习的看法了。

所以我是反对《罗辑思维》第五季的碎片化终生学习法的，您看出来了吧？
